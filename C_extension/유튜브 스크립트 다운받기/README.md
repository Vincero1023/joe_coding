# 유튜브 자막 처리, 요약, 메뉴얼화 (Chrome Extension)

현재 유튜브 영상에서 다음 5가지 작업을 수행하는 크롬 익스텐션입니다.

1. 스크립트(자막) 텍스트 열람
2. 스크립트 텍스트 복사
3. 스크립트 텍스트 다운로드
4. 스크립트 기반 AI 요약 프롬프트 생성
5. 정보 영상/설명 영상을 메뉴얼 문서용 Markdown 프롬프트로 변환

## 기능

- `스크립트 열기`: 자막을 수집해서 확장 내 뷰어 페이지에서 바로 확인
- `스크립트 복사`: 자막 전체를 클립보드로 복사
- `스크립트 다운로드`: 자막을 `.txt` 파일로 저장
- `AI 요약 프롬프트`: 요약용 프롬프트를 생성해 뷰어에 표시하고 AI 사이트 열기
- `AI 메뉴얼 프롬프트`: 설명형 영상을 단계별 메뉴얼로 재구성하는 Markdown 프롬프트 생성
- `Markdown 다운로드`: 뷰어에서 자막, 프롬프트, 디버그 정보를 `.md`로 저장
- `단축키`: `Ctrl+Shift+1/2/3/4` (5번 메뉴얼 기능은 팝업 버튼으로 실행)
- `임베드 인식`: 유튜브 외 사이트의 `youtube embed iframe`도 영상 ID 탐지 시도

## 설치 방법

1. 크롬에서 `chrome://extensions` 열기
2. 우측 상단 `개발자 모드` 켜기
3. `압축해제된 확장 프로그램을 로드합니다` 클릭
4. 이 폴더(`유튜브 스크립트 다운받기`) 선택

## 사용 방법

1. 유튜브 영상 페이지(`https://www.youtube.com/watch?v=...`) 열기
2. 확장 아이콘 클릭
3. 원하는 동작 버튼 선택

## 문제 해결

문제가 생기면 아래 순서로 다시 적용합니다.

1. `chrome://extensions` 열기
2. 기존 압축해제 확장 제거
3. 이 폴더를 다시 `압축해제된 확장 프로그램`으로 로드
4. 팝업 하단 빌드 표시가 `1.5.3-self-healing-fetch-pipeline`인지 확인

## AI 설정

팝업에서 아래 항목을 설정할 수 있습니다.

- `AI 모델`
- `Custom AI URL`
- `자동 전송 여부`
- `ChatGPT Temporary Chat 사용 여부`
- `자막 언어 코드(선택)` 예: `ko`, `en`
- `요약 프롬프트 템플릿 + 프리셋`
- `메뉴얼 프롬프트 템플릿 + 프리셋`

## 메뉴얼화 기능 설명

이 기능은 자막 자체를 메뉴얼로 저장하는 것이 아니라, 자막을 바탕으로 AI가 다시 정리할 수 있도록 전용 프롬프트를 만드는 방식입니다.

- 정보 영상, 튜토리얼, 설명 영상을 단계형 문서로 재구성
- 불필요한 군더더기 제거
- 준비물, 핵심 개념, 단계별 절차, 주의사항, 문제 해결, 체크리스트까지 포함하도록 유도
- 결과를 Markdown 형식으로 반환하도록 설계

## 제한 사항

- 영상에 자막이 없는 경우 스크립트 추출 불가
- 일부 사이트는 CSP/권한 제한으로 임베드 탐지가 실패할 수 있음
- 유튜브 페이지 구조가 크게 바뀌면 파싱 로직 수정이 필요할 수 있음
- 메뉴얼 품질은 원본 자막 품질과 AI 응답 품질의 영향을 받음
