# 상품 정보 추출기 v2.0 - 설치 및 사용 가이드

## 🎉 새로운 기능
- ⌨️ 단축키 지원 추가!
- 🔗 링크 복사 기능 추가
- 📢 화면 알림 표시

## 📥 설치 방법

### 1. 기존 익스텐션 제거 (이미 설치된 경우)
1. Chrome에서 `chrome://extensions/` 접속
2. 기존 "상품 정보 추출기" 찾아서 "삭제" 클릭

### 2. 새 버전 설치
1. 다운로드한 `상품정보추출기_v2.0_단축키지원.zip` 파일 압축 해제
2. Chrome에서 `chrome://extensions/` 접속
3. 우측 상단의 "개발자 모드" 켜기
4. "압축해제된 확장 프로그램을 로드합니다" 클릭
5. 압축 해제한 폴더 선택

## ⌨️ 단축키 사용법

네이버 브랜드 커넥트 상품 상세 페이지에서:

- **Ctrl + Shift + 1** (Mac: Cmd + Shift + 1): 상품명 복사
- **Ctrl + Shift + 2** (Mac: Cmd + Shift + 2): 이미지 URL 복사  
- **Ctrl + Shift + 3** (Mac: Cmd + Shift + 3): 링크 복사

복사가 완료되면 화면 우측 상단에 알림이 표시됩니다!

## 🖱️ 팝업으로 사용하기

1. Chrome 툴바에서 익스텐션 아이콘 클릭
2. 원하는 항목 클릭:
   - 📝 상품명 복사
   - 🖼️ 이미지 URL 복사
   - 🔗 링크 복사

## ⚙️ 단축키 변경 방법

1. `chrome://extensions/shortcuts` 접속
2. "상품 정보 추출기" 찾기
3. 원하는 단축키로 변경

## 🎯 사용 팁

- 단축키는 상품 상세 페이지에서만 작동합니다
- 여러 이미지가 있을 경우 줄바꿈으로 구분되어 복사됩니다
- 복사 완료 후 2초간 알림이 표시됩니다

## ❓ 문제 해결

### 단축키가 작동하지 않을 때
- 페이지 새로고침 (F5)
- 다른 프로그램과 단축키 충돌 확인
- `chrome://extensions/shortcuts`에서 단축키 재설정

### "상품명을 찾을 수 없습니다" 오류
- 네이버 브랜드 커넥트 상품 상세 페이지인지 확인
- 페이지가 완전히 로드될 때까지 대기
