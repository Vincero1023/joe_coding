============================================================
           네이버 블로그 프로 발행기 (사용 설명서)
============================================================

1. 설치 방법
   1) 크롬 주소창에 chrome://extensions 입력
   2) 우측 상단 '개발자 모드' 켜기
   3) '압축해제된 확장 프로그램을 로드합니다' 클릭
   4) 이 폴더 선택

2. 초기 설정 (필수!)
   1) 블로그 글쓰기 화면이나 아무 탭에서 퍼즐 아이콘 -> 확장 프로그램 클릭
   2) [설정 프로필]에서 '+' 눌러서 내 ID 등으로 프로필 생성
   3) 카테고리명(정확히), 시작 시간, 발행 간격 등 입력
   4) [💾 설정 저장] 버튼 클릭 -> "✅ 저장됨" 확인

3. 사용 방법 (두 가지 모드)
   
   [방법 A: 단축키 모드] (추천 👍)
   - 글 작성을 마치면 키보드에서 [ Alt + P ]를 누르세요.
   - 자동으로 발행 버튼을 찾아서 누르고, 다음 예약 시간을 계산해 입력해줍니다.
   - 우측 상단에 검은색 알림창이 뜨면 확인 후 최종 '발행' 클릭.

   [방법 B: 버튼 클릭 모드]
   - 확장 프로그램 아이콘을 누르고 [🚀 발행 시작] 버튼 클릭.

4. 주요 기능
   - 프로필 관리: 부계정 등 여러 설정을 저장해두고 불러올 수 있음.
   - 랜덤 간격: 60분 설정 시 ±20% (48분~72분) 사이로 랜덤하게 잡아줌 (어뷰징 방지).
   - 금지 시간: 새벽 시간 등 발행을 원치 않는 시간대를 설정하면 자동으로 건너뜀.
   - 휠 조작: 팝업창 시간 입력칸 위에서 마우스 휠을 굴리면 1분 단위 조절 가능.

5. 문제 해결
   - "버튼을 찾을 수 없습니다": 글쓰기 에디터가 완전히 로딩된 후 시도해주세요.
   - 시간이 이상해요: [🔄 시간 리셋] 버튼을 눌러 초기화하세요.
