# Agoda Best 2.0

Agoda 호텔 상세 페이지에서 여러 캠페인 링크를 빠르게 비교하고, 예약한 가격까지 추적할 수 있는 Chrome 확장 프로그램입니다.

## 핵심 기능

- 현재 Agoda 호텔 페이지를 기준으로 여러 `cid/tag` 조합을 스캔합니다.
- 카드사, 검색 경로, 항공사 적립 링크를 함께 비교합니다.
- `desktop` / `mobile` 가격 변형을 같이 확인합니다.
- 결과 탭에서 상위 가격, 전체 비교 결과, 바로 열기 링크를 확인합니다.
- 예약 URL과 예약 가격을 저장하고, 이후 가격 하락 여부를 다시 점검합니다.

## 주요 화면

- `popup.html`: 스캔 시작, 옵션 선택, 결과/예약 추적기로 이동
- `results.html`: 실시간 스캔 결과, 상위 3개 결과, 예약 저장
- `reservations.html`: 저장한 예약 목록, 수동 재확인, 최저가 다시 열기

## 사용 방법

1. `chrome://extensions`에서 개발자 모드를 켭니다.
2. 이 폴더를 압축 해제된 확장 프로그램으로 불러옵니다.
3. Agoda 호텔 상세 페이지를 엽니다.
4. `Agoda Best` 팝업에서 비교할 옵션을 선택합니다.
5. `상위 3개 검색` 또는 `검색 후 1위 열기`를 실행합니다.
6. 결과 탭에서 최저가 링크를 열거나 예약 추적 목록에 저장합니다.

## 예약 추적

- 결과 탭의 `예약 저장` 버튼으로 현재 페이지를 저장할 수 있습니다.
- 예약 추적기에서 Agoda URL과 예약 가격을 직접 입력해 등록할 수도 있습니다.
- 저장 가능 개수는 최대 5개입니다.
- 24시간마다 자동 점검하며, 더 저렴한 가격이 발견되면 알림을 띄웁니다.
