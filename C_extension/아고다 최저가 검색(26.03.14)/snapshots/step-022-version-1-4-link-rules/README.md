# Agoda Campaign Scanner 1.0

현재 Agoda 숙소 상세 페이지를 기준으로, 미리 정의된 `cid` 규칙과 제휴 프로모션 규칙을 순차 비교하는 크롬 확장 프로그램입니다.

## 핵심 동작

- 현재 호텔 상세 URL의 날짜, 인원, 객실 조건을 유지한 채 선택된 규칙만 비교합니다.
- 검색을 시작하면 팝업 대신 결과 전용 탭이 열리고, 그 탭에 진행 중 결과와 완료 결과가 계속 남습니다.
- 일반 캠페인 규칙은 호텔 상세 페이지 가격을 추출해서 비교합니다.
- `제휴 프로모션` 카테고리는 일반 프로모션 페이지라서 가격 비교 대상이 아니라 `열기 전용`으로 취급합니다.

## 카테고리

- 검색 경로: 구글 검색 A/B/C, 구글지도 A/B/C, 네이버 검색
- 카드/결제: BC, 국민, 마스터카드, 비자, 신한(마스터), 신한카드, 토스, 현대카드, 하나, 우리(마스터), 우리카드, 유니온페이, 카카오페이
- 항공사/적립: 대한항공(적립), 아시아나항공(적립), 에어서울
- 제휴 프로모션: 일반 프로모션 페이지 바로가기

## 사용 방법

1. `chrome://extensions`에서 개발자 모드를 켭니다.
2. 이 폴더를 압축해제된 확장 프로그램으로 로드합니다.
3. Agoda 숙소 상세 페이지를 엽니다.
4. 팝업에서 원하는 규칙을 체크합니다.
5. `Top 3 검색` 또는 `검색 후 1위 열기`를 누릅니다.
6. 열리는 결과 탭에서 진행 상태, 실패 사유, 링크 열기를 확인합니다.

## 제휴 프로모션 주의

- 현재 입력된 제휴 프로모션 URL은 사용자가 준 템플릿 문자열 그대로 들어가 있습니다.
- `https://www.agoda.com/.../something-promo` 형태의 `...`는 실제 Agoda 경로로 바꿔야 정상 동작합니다.
- 실제 URL이 들어오기 전까지 결과 탭에서는 `실제 URL 필요` 상태로 표시됩니다.
