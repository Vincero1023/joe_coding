# Agoda Campaign Scanner

현재 Agoda 숙소 상세 페이지를 기준으로, 미리 정의한 `cid`/`tag` 캠페인 링크를 자동으로 열어보고 가격을 추출해 Top 3를 보여주는 크롬 익스텐션입니다.

## 핵심 동작

- 현재 숙소 페이지의 날짜, 인원, 객실 수 같은 검색 조건은 유지합니다.
- 제공받은 링크를 분석해 실질적으로 달라지는 `cid`와 일부 `tag`를 카탈로그로 구성했습니다.
- 검색 시 `ko-kr`, `KRW`, `userCountry=KR`, `finalPriceView=1` 기준으로 각 캠페인 링크를 생성합니다.
- 결과는 배경 탭에서 순차 스캔하고, 추출된 가격 기준으로 Top 1, 2, 3을 보여줍니다.
- 카드/결제 카테고리는 보유 카드만 체크해두면 해당 카드 링크만 포함해 검색합니다.

## 포함된 카테고리

- 기본/유입: 기본, 구글 지도 1~3, 구글 검색 1~3, 네이버 검색 1
- 카드/결제: 현대, 국민, 유니온페이, 카카오페이, 우리, BC, 신한, 토스, 하나, 마스터, 비자 등
- 항공사: 대한항공, 아시아나, 에어서울

## 사용 방법

1. `chrome://extensions`에서 `개발자 모드`를 켭니다.
2. `압축해제된 확장 프로그램을 로드합니다`.
3. 이 폴더를 선택합니다.
4. Agoda 숙소 상세 페이지를 엽니다.
5. 익스텐션 팝업에서 원하는 카드와 소스를 체크합니다.
6. `Top 3 검색` 또는 `검색 후 1위 열기`를 누릅니다.

## 현재 한계

- Agoda가 페이지 구조를 바꾸면 가격 추출 규칙도 다시 맞춰야 합니다.
- 일부 링크는 실제 할인 대신 같은 가격을 돌려줄 수 있습니다.
- 카드 프로모션이 결제 단계에서만 확정되면 숙소 페이지에서 차이가 안 보일 수 있습니다.
- 공용 소비자 사이트 특성상 IP 기반 국가 가격 차이는 이 익스텐션만으로 완전히 재현하기 어렵습니다.
