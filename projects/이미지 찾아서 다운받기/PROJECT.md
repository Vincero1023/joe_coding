# Project Overview

## Purpose
- 특정 사이트에서 필요한 이미지를 반복적으로 찾고 내려받는 수작업을 자동화한다.
- 다운로드한 이미지를 규칙에 맞는 폴더 구조로 정리해 후속 작업 시간을 줄인다.

## Features
- 키워드나 목록 입력을 기준으로 대상 이미지를 검색하고 수집
- 다운로드한 이미지를 이름 규칙과 폴더 구조에 맞게 저장
- 실패, 중복, 누락 항목을 확인할 수 있는 실행 로그 제공

## Tech Stack
- Python
- HTTP 요청 및 HTML 파싱 라이브러리 (`requests`, `BeautifulSoup4`) 예정
- 파일 처리용 표준 라이브러리와 테스트용 `pytest` 예정

## Development Plan
1. 입력 형식과 대상 사이트 구조를 정의하고 최소 동작 흐름을 설계한다.
2. 검색, 다운로드, 저장 경로 분류 로직을 구현한다.
3. 실패 케이스, 중복 처리, 테스트와 실행 로그를 보강한다.

## Idea
반복적인 이미지 수집 작업을 한 번의 입력으로 검색, 다운로드, 폴더 정리까지 끝내는 자동화 도구
