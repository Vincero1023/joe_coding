필터 사용법 (초보자 가이드)
PC조회↑ = PC에서 이 키워드를 한 달간 검색한 횟수의 최솟값
예) 100 입력 → PC 월 검색량이 100 이상인 키워드만 수집
MO조회↑ = 모바일에서 한 달간 검색한 횟수의 최솟값
예) 500 입력 → 모바일 월 검색량이 500 이상인 것만
합계↑ / 합계↓ = PC+MO 합산 검색량의 최솟값 / 최댓값
예) 합계↑ 100, 합계↓ 5000 → 월 검색량 100~5000 사이만 수집
팁: 합계↑에 1만 넣어도 검색량 0인 키워드가 걸러집니다!
블로그↑ = 네이버 블로그 글 수의 최솟값
예) 10 입력 → 관련 블로그 글이 10개 이상인 키워드만
PC클릭↑ / MO클릭↑ / 클릭합↑ = 월간 평균 클릭 수의 최솟값
예) 클릭합↑ 50 → 월 PC+MO 클릭이 50회 이상인 것만
입찰1↑ / 입찰2↑ / 입찰3↑ = 네이버 검색광고 입찰가(원)의 최솟값
예) 입찰1↑ 500 → 1위 입찰가가 500원 이상인 키워드만
입찰가가 높을수록 광고주들이 돈을 많이 쓰는 = 돈이 되는 키워드!
추천 세팅: 처음이면 합계↑ = 1만 입력하세요. 검색량이 아예 0인 쓸모없는 키워드만 걸러줍니다.
