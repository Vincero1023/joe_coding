# 키워드마스터 사용법 완벽 가이드 초보도 5분이면 끝

## 키워드마스터란?

키워드마스터는 네이버 키워드 검색량 조회 와 CPC 입찰가 분석 을 한 번에 할 수 있는 무료 키워드 분석 도구입니다. 블로그를 운영하든, 네이버 광고를 집행하든, 키워드 리서치의 첫 단계는 항상 "이 키워드가 얼마나 검색되는가?"를 파악하는 것입니다. 키워드마스터는 이 과정을 빠르고 정확하게 도와줍니다.

기존 네이버 키워드 도구(검색광고 시스템)는 광고주를 위한 도구이기 때문에 블로거나 마케터가 사용하기에는 불편한 점이 많습니다. 키워드마스터는 블로거와 마케터에게 최적화된 인터페이스 로 설계되어, 누구나 쉽게 키워드 데이터를 분석할 수 있습니다.

## 5분 만에 배우는 기본 사용법

키워드마스터를 처음 사용하는 분이라도 아래 단계를 따라 하면 5분 안에 모든 핵심 기능을 익힐 수 있습니다.

- 키워드마스터 접속 — keywordmaster.net 에 접속합니다. 별도 회원가입 없이 바로 사용 가능합니다.
- 키워드 입력 — 좌측 상단 검색창에 분석하고 싶은 키워드를 입력하고 검색 버튼을 누릅니다. 예: "강남 맛집"
- 검색 결과 확인 — 검색 즉시 PC 조회수, 모바일 조회수, 총 조회수, PC/MO 클릭수, 블로그 발행량이 한눈에 표시됩니다.
- CPC 입찰가 분석 — 검색 결과 아래에서 PC 입찰가(1~10위)와 MO 입찰가(1~5위)를 차트로 확인합니다. 입찰가가 높을수록 광고주에게 인기 있는 = 돈이 되는 키워드입니다.
- 연관 키워드 탐색 — 메인 키워드 아래에 연관 키워드 목록이 등급별(S~F)로 표시됩니다. 점수가 높은 키워드를 클릭하면 해당 키워드로 다시 분석됩니다.
## 핵심 기능 상세 안내

### 스코어 등급 시스템 (S~F)

키워드마스터는 모든 키워드에 스코어 등급 을 부여합니다. 스코어는 CPC(40%) + 검색량(35%) + 희귀성(25%)으로 계산되며, S등급(85점 이상)부터 F등급(25점 미만)까지 6단계로 나뉩니다.

- S등급 (85+) — 최고 가치 키워드. CPC가 높고, 검색량이 많고, 블로그 글이 적은 황금 키워드
- A등급 (70+) — 높은 가치. 적극적으로 공략할 만한 키워드
- B등급 (55+) — 양호. 포트폴리오에 포함할 만한 키워드
- C~F등급 — 일반적이거나 가치가 낮은 키워드
### 대량 수집 & 분석

좌측 사이드바의 "대량 수집 & 분석" 기능은 키워드마스터의 핵심 기능 중 하나입니다. 시드 키워드를 입력하면 연관 키워드와 자동완성 키워드를 자동으로 확장 수집하여, 한 번에 1,000개에서 최대 무제한까지 키워드를 수집할 수 있습니다.

- 연관확장 — 네이버 연관 키워드를 자동으로 추가 수집합니다
- 자동완성 — 네이버 자동완성 데이터를 활용해 키워드를 확장합니다
- 원문포함 — 원래 검색어를 포함하는 키워드만 필터링합니다
### 필터 옵션

수집된 키워드를 조건에 맞게 걸러낼 수 있습니다. PC조회, MO조회, 합계, 블로그 수, 클릭 수, 입찰가 등 다양한 필터를 설정하여 원하는 조건의 키워드만 추출할 수 있습니다.

### 키워드 조합기

지역, 업종, 의도, 장소, 시기의 5개 슬롯에 키워드를 넣으면 자동으로 모든 조합을 생성합니다. 예를 들어 지역에 "강남, 홍대", 업종에 "맛집, 카페"를 넣으면 "강남 맛집", "강남 카페", "홍대 맛집", "홍대 카페" 4개 조합이 자동 생성됩니다.
