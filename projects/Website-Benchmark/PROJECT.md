# Project Overview

## Purpose
- 웹사이트를 분석해 핵심 기능과 화면 구조를 빠르게 파악한다.
- 벤치마킹과 재구현에 필요한 정보를 문서 형태로 정리한다.

## Features
- 주요 페이지와 섹션 구조 분석
- 핵심 기능 및 UI 패턴 추출
- 공통 컴포넌트와 사용자 흐름 정리
- 분석 결과를 Markdown 문서로 기록
