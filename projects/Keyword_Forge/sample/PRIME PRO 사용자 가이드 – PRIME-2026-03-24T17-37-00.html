PRIME PRO 사용자 가이드

2026.03.10 16:40 관리자

프로그램 설치 및 압축플기 시 터미널이 잠깐 깜빡이는 건 정상입니다. ( 무단 사용 및 불법 사용자 체킹용 작용 )

프로그램을 실행하면 로그인 화면이 먼저 표시됩니다. 발급받은 마스터키를 입력한 뒤 로그인 버튼을 눌러
접속합니다, 로그인 완료 후 기본 화면은 대시보드입니다.

마스터키 오류나 발급 문제는 관리자 개인 톡 부탇드립니다. -> 관리자 1대1 개인 카톡

맥주소와 기기값은 자동 저장 되며, 최대 5개의 기기만 등록이 가능합니다, 초과 등록 시 초과 등록한 기기는
로그아웃 됩니다, 또한 맥주소를 임의로 변경 하여 프로그램을 가동시킬 경우, 우회 시도가 있을 경우

이유를 불문 하고 즉시 프로그램 영구 차단 조치 및 손해배상 청구가 이루어 질 수 있습니다.

주의사항

네이버 계정은 다 계정 포스팅시에 꼭 고정 프록시를 사용하셔야 합니다!
( 계정이 2개가 넘어갈 경우 꼭 프록시를 적용하여 작성 해 주시길 바랍니다 1계정 1프록시 or 2계정 1프록시 )

쇼커넥/쿠파스 포스팅시 공정위 문구를 직접 지정하여 커스텀 하시는 게 좋습니다.
( 10개 이상 공정위 이미지 넣고 랜덤으로 돌리는게 베스트 )

네이버 계정은 디자인 설정에서 텍스트 색상 및 테두리 디자인 커스텀/인용구 커스텀 하시는 게 좋습니다.

로그인을 하게 되면 위 이미지와 같이 대시보드에 들어오게 됩니다.

사용자가 구매한 플랫폼에 맞춰 특정 기능들은 잠금이 되어있고 구매한 플랫폼 관련 기능은 열려있으므로,
구매한 플랫폼이 잘 열려있는지 확인 해 보시길 바랍니다.

작업 생성 내역


작업 생성 내역은 자동화 작업들의 현재 진행 상태를 볼 수 있는 섹션 입니다. 자동화 작업을 실행 하게 되면
작업 생성 내역 섹션에 위 이미지와 같이 진행 상황 바가 추가되게 됩니다.

실패/성공/진행중 등등 상태 뱃지가 업데이트 되고 우측 X 눌러 삭제/중단을 하실 수 있습니다.

작업 상세 로그


작업 상세 로그에서는 각 작업의 상세 로그를 보실 수 있습니다, 우측 상단 드롭다운 박스를 눌러 작업을 전환하여
다른 작업의 로그를 보실 수 있고 복사 및 삭제가 가능합니다.

편의 대시보드


편의 대시보드에서는 네이버 QR 한도와 자동화 매니저에서 작업하는 자동화 봇의 로그를 보실 수 있습니다.
( 해당 기능은 아래에서 다시 안내드리겠습니다. )

설정

설정은 자동화에 필요한 모든 계정, API 키, 프록시 정보를 등록 하는 메뉴입니다.

꼭 자동화를 실행하기 전 설정에서 계정/API/프록시를 먼저 등록 하셔야 합니다.

중요 안내

  • 설정이 비어 있으면 자동화 실행이 진행되지 않습니다.
  • 필요한 플랫폼을 먼저 저장한 뒤 자동화 페이지에서 선택해야 합니다.

Chat Gemini/Chat Claude 연결은


1. Node.js LTS 설치

  • 공식 사이트로 이동: Node.js
  • LTS 버전 다운로드
  • 설치 마법사에서 기본값으로 계속 진행
  • 설치 후 새 PowerShell 열고 확인:

node -v npm -v

2. Git 설치

  • 공식 사이트로 이동: Git for Windows
  • 설치 파일 실행
  • 특별히 바꿀 거 없으면 기본값으로 진행
  • 설치 후 새 PowerShell 열고 확인:

git --version

3. 설치 후

PRIME 프로그램 완전히 종료 까지 하시고 다시 켜서 연결하셔야 합니다!

플랫폼별 입력 가이드

  • 네이버 계정 = 네이버 자동화 작업을 진행 할 아이디/비밀번호 입력
    -> 네이버 블로그, 네이버 카페 등 공통 사용
  • 워드프레스 = 워드프레스 URL 입력
    -> 폴더안의 프라임 워프 플러그인을 워드프레스에 설치 후 URL 입력해 주셔야 자동화 가능
  • 티스토리 = 아이디/비밀번호/블로그 주소 입력
    -> 블로그 주소는 꼭 tistory.com 형식 주소로 적어 주셔야 합니다.
  • 블로그 스팟 = 블로그 스팟 연결 버튼으로 OAuth 연동
    -> 연결 버튼을 누르면 자동으로 새창이 뜨게 되며 해당 창에서 블로그 스팟 구글 계정으로 로그인
  • 쿠팡 파트너스 = API 키 입력
  • Chat GPT 연결 = 연결 버튼으로 계정 연동
    -> 연결 버튼을 누르면 자동으로 생창이 뜨게 되며 해당 창에서 GPT 계정으로 로그인
  • Chat Gemini 연결 = 연결 버튼으로 계정 연동
    -> 연결 버튼을 누르면 자동으로 생창이 뜨게 되며 해당 창에서 Gemini 계정으로 로그인
  • Chat Claude 연결 = 연결 버튼으로 계정 연동
    -> 연결 버튼을 누르면 자동으로 생창이 뜨게 되며 해당 창에서 Claude 계정으로 로그인
  • 로컬 LLM 연결 -> 연결 버튼으로 로컬 LLM 다운로드
    -> 연결 버튼을 누르면 프로그램이 자동으로 LLM 다운로드 후 성공 메세지
  • 쓰레드 = 유저 ID + 액세스 토큰 입력
  • 네이버 카페 정보 = 카페 URL + 게시판 이름 입력
  • 프록시 설정 = 프록시 IP + 포트 입력
  • AI 설정 = OpenAI/Gemini/Claude/OpenRouter/Pixabay 키 입력
  • 네이버 API = 검색광고/데이터랩 키 입력
  • 자동 색인 = 네이버 색인 계정 입력 + 구글 계정 연동

설정에서는 계정들을 일괄 붙여넣기 여러 계정을 줄바꿈으로 한번에 붙여넣기 할 수 있습니다.

  • 예시:
    아이디 비번
    아이디 비번
    아이디 비번

꼭 입력후 설정 저장 하기를 눌려야 저장이 됩니다. 꼭 저장 후 연동된 계정 정보에서 등록 상태 확인 해주세요.

저장을 잘 하셨으면 연동 된 계정 정보에 저장된 값이 뜨게 됩니다.

블로그 자동화

블로그 자동화 메뉴는 워드프레스/티스토리/블로그 스팟/네이버 블로그 등 ” 블로그 ” 플랫폼들을 자동화 할 수 있는 메뉴 페이지 입니다.

설정에서 저장한 값들을 각 행/표에서 선택하고 다양한 기능들을 On Off 선택하여 자동화를 돌리실 수 있습니다.

블로그 자동화 기본 흐름

1) 포스팅 진행할 플랫폼 선택
2) 포스팅 진행할 계정만큼 행 추가 및 해당 행들 설정
3) 기능에 맞는 세팅 및 추가 확인
4) 선택 실행/포스팅 미리보기 또는 전체 실행
5) 대시보드로 이동 후 로그 및 진행 상황 확인

블로그 자동화 필드 설명 (행 기준)

  • 포스팅 계정 = 설정에서 저장한 해당 플랫폼 계정 선택
  • 포스팅 모드 = 일반/크롤링/쇼커넥/쿠파스 중 선택
  • 포스팅 주제 = 모드별 주제 폴더 파일 선택
  • 발행 설정 = 임시 저장/즉시 발행/예약 발행
  • AI 모델 선택 = 연동된 AI 중 선택
  • 발행 간격/예약 간격 = 최소/최대값 슬라이더로 조절
  • 프롬프트 선택 = 글생성 프롬프트 파일 선택
  • 링크 설정 = 포스팅에 삽입할 링크 파일 선택
  • 내부링크 설정 = 본문 내 내부 링크 파일 선택
  • 썸네일/중간 이미지 = 이미지 설정 폴더 리소스 선택
  • 디자인 설정 = 시스템 디자인 설정 프리셋 적용
  • 사용 프록시 = 없음/계정 기반/사용자 지정 프록시 선택

모드별 주제 폴더 안내

  • 일반 모드: 포스팅 주제/주제
  • 크롤링 모드: 포스팅 주제/크롤링 주제
  • 쇼커넥 모드: 포스팅 주제/쇼핑커넥트 주제
  • 쿠파스 모드: 포스팅 주제/쿠팡 파트너스 주제
  • 빠른 확인: 빠른 폴더 작업 또는 시스템 에이전트 활용

실행 버튼

  • 선택 행 포스팅 시작: 체크 행만 실행
  • 하단 고정 포스팅 시작하기: 현재 구성 전체 실행

결과 확인

  • 대시보드 작업 생성 내역/상세 로그

주의사항

예약 발행은 예약 간격 설정까지 확인해야 합니다. 필수값(계정/모드/주제/AI) 누락 시 실행되지 않습니다.

자세한 각 폴더 확인은 빠폴작 -> 빠른 폴더 작업에서 확인 하실 수 있습니다.

외부유입 자동화

외부유입 자동화 메뉴는 쓰레드 또는 네이버 카페 등등 외부유입 플랫폼들을 자동화 할 수 있는 메뉴 페이지입니다.

외부유입 자동화 기본 흐름

1) 포스팅 진행할 플랫폼 선택
2) 포스팅 진행할 계정만큼 행 추가 및 해당 행들 설정
3) 기능에 맞는 세팅 및 추가 확인
4) 선택 실행/포스팅 미리보기 또는 전체 실행
5) 대시보드로 이동 후 로그 및 진행 상황 확인

외부유입 자동화 필드 설명

  • 포스팅 계정 = 설정에 저장된 계정 선택
  • 포스팅 모드 = 일반/크롤링/쇼커넥/쿠파스(쓰레드는 쿠파스 유머 포함)
  • 포스팅 주제 = 모드별 파일 선택
  • AI 모델 선택 = 연동된 모델 선택
  • 발행 간격 = 최소/최대 설정
  • 프롬프트 선택 = 사용할 프롬프트 파일
  • 링크 설정 = 첨부 링크 파일
  • 디자인 설정 = 네이버 카페 사용 시 적용 가능
  • 사용 프록시 = 프록시 방식 선택

네이버 카페 자동화시 필수 설정

  • 네이버 카페 플랫폼은 카페 정보 선택이 필요합니다. ( 설정에서 저장하실 수 있습니다. )

실행 버튼

  • 선택 행 작업 시작: 체크 행만 실행
  • 하단 고정 포스팅 시작하기: 전체 실행

결과 확인

  • 대시보드 작업 생성 내역/상세 로그

주의사항

  • 플랫폼별 지원 모드가 다를 수 있으므로 모드 선택 시 드롭다운 항목을 기준으로 설정합니다.

에이전트 자동화 (준비중)

목적

  • 향후 자동 규칙 실행 기능 영역

현재 상태

  • 안내 화면만 제공되며 실제 운영 기능은 추후 업데이트 예정

자동화 매니저

자동화 매니저 메뉴는 시스템 설정 및 디자인, 플랫폼 별 크롤러 및 봇, 자동 색인 작업등등 부가적인
자동화 도구들을 한 화면에서 작업 하실 수 있는 메뉴 페이지 입니다.

자동화 매니저 기본 흐름

1) 좌측 탭 선택
2) 행 설정 또는 토글 설정
3) 기능에 맞는 세팅 및 추가 확인
4) 실행 및 저장
5) 대시보드로 이동 후 로그 및 진행 상황 확인

시스템 자동화 설정

  • 용도: 전역 자동화 동작 제어
  • 주요 항목: 네이버 연관 키워드, 포스팅 집중 모드, 매니저 순차 실행, 금칙어 파일, 포스팅 딜레이
  • 추가 기능: 중복 이력 전체 초기화 버튼

네이버 API 연관 키워드 -> 네이버 API 설정 저장 필요, 포스팅 시 연관 키워드 분석 후 삽입
포스팅 집중 모드 -> 포스팅 작업 시 절전/보호기 작용을 막아줌
매니저 크롤링 순차 실행 -> 크롤링 잡업시 병렬 처리가 아닌 순차 처리로 크롤링 해줌
금칙어 변환 파일 -> 정보 폴더에 저장한 금칙어 txt 에 맞게 금칙어를 금지 및 변환 해줌
크롤링 금칙어 파일 -> 쇼핑 커넥트/쿠팡 파트너스 크롤링 시 금칙어를 적용함
포스팅 딜레이 -> 특정 시간만 포스팅 진행/특정 글갯수 작성시 딜레이 추가 가능

쇼핑 커넥트/쿠팡 이력 중복 제거 -> 쇼커넥/쿠팡 크롤링시 중복 크롤링을 제외함

시스템 디자인 설정

  • 용도: 이미지/인용구/후킹 문구 전역 스타일 설정
  • 주요 항목: 고정 이미지, 공정위 이미지, 텍스트 색상, 테두리 스타일, 후킹 문구
  • 저장 버튼: 설정 저장하기

고정 이미지/위치 -> 고정 이미지를 선택하고 위치를 정하면 해당 위치에 이미지 삽입
공정위 문구 이미지/패턴 -> 공정위 문구 이미지를 기본/사용자 지정 이미지 중 커스텀
네이버 이모티콘/구분선/인용구/포인트 -> 네이버 블로그/카페 포스팅 시 적용 되는 디자인

썸네일 텍스트/중간 이미지 텍스트/텍스트 테두리 -> 썸네일/중간 이미지에 들어가는 텍스트 커스텀

이미지 테두리 디자인/색상 -> 모든 이미지에 적용되는 테두리 디자인과 색상 커스텀

필수 설정 저장

  • 설정 > AI 설정: 글 생성/문구 생성 작업 필수
  • 설정 > Chat AI 연결: 구독제 AI 계정 연동이 필요한 작업 필수
  • 설정 > 프록시 설정: 프록시를 사용할 작업이면 먼저 저장

필수 폴더/파일

  • 프롬프트 선택/글생성 프롬프트: 작업별 프롬프트 .txt
  • 정보/링크: 외부/내부 링크 .txt
  • 포스팅 주제/*: 모드별 주제 파일

실행 전 점검
1) 설정 저장하기 클릭 후 연동된 계정 정보에 실제로 보이는지 확인
2) 행에서 드롭다운을 열었을 때 계정/파일 항목이 비어 있지 않은지 확인
3) 비어 있으면 빠른 폴더 작업에서 파일 생성 후 다시 진입

블로그 자동화

필수 설정 저장

  • 플랫폼별 계정 중 사용하는 플랫폼을 반드시 저장
    네이버 블로그 = 네이버 계정
    워드프레스 = 워드프레스 URL(연동 준비 완료 상태)
    티스토리 = 티스토리 아이디/비밀번호/블로그 주소
    블로그스팟 = 블로그 스팟 연결(OAuth)
  • AI 설정 또는 Chat GPT 연결 중 실제 사용 모델 기준으로 준비
  • 프록시 사용 시 프록시 설정 저장

필수 폴더/파일

  • 주제 파일:
    일반 = 포스팅 주제/주제
    크롤링 = 포스팅 주제/크롤링 주제
    쇼커넥 = 포스팅 주제/쇼핑커넥트 주제
    쿠파스 = 포스팅 주제/쿠팡 파트너스 주제
  • 프롬프트 파일 = 프롬프트 선택/글생성 프롬프트
  • 링크 파일 = 정보/링크 (모드별 링크 파일 권장 분리)
  • 이미지 파일 = 이미지 설정/썸네일 이미지, 이미지 설정/중간 이미지

실행 전 점검
1) 행에서 포스팅 계정/모드/주제/AI 모델 선택 완료
2) 예약 발행이면 예약 간격까지 지정
3) 포스팅 미리보기로 결과 점검 후 실행

외부유입 자동화 (네이버 카페)

필수 설정 저장

  • 설정 > 네이버 계정
  • 설정 > 네이버 카페 정보 (카페 URL + 게시판 이름)
  • 설정 > AI 설정 또는 Chat GPT 연결
  • 필요 시 설정 > 프록시 설정

필수 폴더/파일

  • 주제 파일 = 포스팅 모드에 맞는 주제 폴더
  • 프롬프트 파일 = 프롬프트 선택/글생성 프롬프트
  • 링크 파일 = 정보/링크
  • 크롤링/상거래 모드 사용 시 = 정보/쇼핑커넥트, 정보/쿠팡 파트너스 데이터 확인

실행 전 점검
1) 플랫폼을 네이버 카페로 선택
2) 카페 정보 드롭다운에서 대상 카페 선택
3) 선택 행 실행 또는 전체 실행

외부유입 자동화 (쓰레드)

필수 설정 저장

  • 설정 > 쓰레드 (유저 ID + 액세스 토큰)
  • 설정 > AI 설정 또는 Chat GPT 연결
  • 쿠파스/쇼커넥 연계 모드 사용 시 설정 > 쿠팡 파트너스도 저장
  • 필요 시 설정 > 프록시 설정

필수 폴더/파일

  • 일반/크롤링 모드 = 포스팅 모드에 맞는 주제 폴더
  • 쿠파스 유머/커머스 모드 = 포스팅 주제/쓰레드 쿠팡 파트너스 주제
  • 프롬프트 파일 = 프롬프트 선택/글생성 프롬프트
  • 링크 데이터 = 정보/쿠팡 파트너스 또는 정보/쇼핑커넥트
  • 미디어 보조 데이터 = 정보/쓰레드쿠팡_미디어

실행 전 점검
1) 포스팅 모드와 주제 파일이 서로 맞는지 확인
2) 링크 기반 댓글이 필요한 모드면 소스 파일에 실제 링크가 있는지 확인
3) 링크가 없는 항목은 건너뛰도록 로그를 확인하며 실행

자동화 매니저 – 탭별 준비 항목

시스템 자동화 설정

  • 필수 설정 저장: 없음(내부 전역 설정)
  • 필수 폴더/파일: 금칙어 파일 사용 시 금칙어 파일 준비
  • 점검: 설정 저장 후 실제 자동화 작업에 반영되는지 로그로 확인

시스템 디자인 설정

  • 필수 설정 저장: 없음(디자인값 자체 저장)
  • 필수 폴더/파일: 고정 이미지 사용 시 이미지 파일 준비
  • 점검: 설정 저장하기 후 미리보기/실행 결과에서 디자인 반영 확인

사이트맵 크롤러

  • 필수 설정 저장: 필요 시 프록시 설정
  • 필수 폴더/파일: 크롤링 링크 파일, 저장 위치
  • 점검: 파일 선택 드롭다운에 소스 파일이 보이는지 확인

쇼핑 커넥트 크롤러

  • 필수 설정 저장: 네이버 계정
  • 필수 폴더/파일: 검색 파일, 저장 위치
  • 점검: 가격대/상품 수/속도 입력 후 실행

쿠팡 파트너스 크롤러

  • 필수 설정 저장: 쿠팡 파트너스 API
  • 필수 폴더/파일: 검색/URL 파일, 저장 위치
  • 점검: 계정 선택, 파일 선택, 프록시(필요 시) 확인

쓰레드 쿠팡 크롤러

  • 필수 설정 저장: 쿠팡 파트너스 API(변환 사용 시)
  • 필수 폴더/파일: 검색/URL 파일, 저장 위치
  • 점검: 쓰레드 쿠팡 작업 시작 전 연결 환경 상태 확인

네이버 카페 봇

  • 필수 설정 저장: 네이버 계정, 네이버 카페 정보
  • 필수 폴더/파일: 소스 파일(게시글/댓글 소스), 저장 위치
  • 점검: 계정/카페 정보/활동 시간대 설정 후 시작

네이버 서이추 봇

  • 필수 설정 저장: 네이버 계정
  • 필수 폴더/파일: 검색 파일, 메시지 파일
  • 점검: 대상 수/프록시 설정 확인 후 시작

네이버 계정 대시보드

  • 필수 설정 저장: 네이버 계정
  • 필수 폴더/파일: 없음
  • 점검: 새로고침으로 계정별 통계 확인

네이버 지수 매니저

  • 필수 설정 저장: 네이버 계정
  • 필수 폴더/파일: 없음
  • 점검: 진단 대상 계정 상태 확인 후 재진단

자동 색인 매니저

  • 필수 설정 저장: 자동 색인 계정(네이버), 구글 계정 연결
