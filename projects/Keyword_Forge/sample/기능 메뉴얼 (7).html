키워드마스터
완벽 매뉴얼

10가지 핵심 기능을 탭별로 정리했습니다. 초보자부터 고급 사용자까지, 필요한 기능을 빠르게 찾아보세요.

1

키워드 검색 기본

네이버 키워드 검색량과 CPC를 한눈에 파악하세요

1

검색창에 키워드 입력

상단 검색창에 분석하고 싶은 키워드를 입력하고 Enter 또는 검색 버튼을 클릭합니다. 예: 다이어트 식단

2

검색 결과 확인

입력한 키워드의 PC 검색량 모바일 검색량 총 검색량이 즉시 표시됩니다. 연관 키워드도 함께 제공됩니다.

3

CPC(클릭당 비용) 분석

각 키워드의 예상 입찰가와 예상 클릭수를 확인할 수 있습니다. 광고비 대비 효율이 높은 키워드를 찾아보세요.

💡 Tip: 연관 키워드를 클릭하면 해당 키워드로 바로 재검색됩니다. 키워드 아이디어를 확장해 나가세요!
키워드 검색 바로가기 →
2

커스텀 키워드 세트

여러 키워드를 그룹으로 묶어서 한 번에 관리하세요

1

커스텀 키워드 입력창 열기

검색 페이지 왼쪽의 커스텀 키워드 영역에서 분석하고 싶은 키워드를 한 줄에 하나씩 입력합니다. 콤마(,)로 구분해도 됩니다.

2

세트 추가 버튼 클릭

세트 추가 버튼을 클릭하면 입력한 키워드들이 하나의 분석 세트로 생성됩니다. 여러 개의 세트를 만들어 주제별로 관리할 수 있습니다.

3

세트 내 키워드 클릭으로 개별 분석

세트에 추가된 키워드를 하나씩 클릭하면 해당 키워드의 상세 데이터가 오른쪽 결과 영역에 표시됩니다.

🎯 활용 예시: "카페 창업" 세트에는 카페 창업 비용, 카페 인테리어, 카페 메뉴판 등을 넣고, "맛집" 세트에는 강남 맛집, 홍대 맛집, 부산 맛집 등을 넣어 주제별로 분류하세요.
검색 페이지에서 시도하기 →
3

자동 분석 기능

수집한 키워드를 자동으로 순차 분석합니다

1

키워드 수집

먼저 키워드를 검색하여 연관 키워드를 수집하거나, 커스텀 세트에 키워드를 직접 입력합니다.

2

자동 분석 시작

세트 상단의 a 버튼을 클릭하면, 목록의 키워드를 하나씩 자동으로 검색하며 데이터를 수집합니다.

3

결과 확인

자동 분석이 완료되면 모든 키워드의 검색량, CPC, 블로그 발행량 데이터가 테이블에 채워집니다. 이 데이터를 기반으로 황금 키워드를 선별하세요.

⚠️ 주의: 자동 분석 중에는 페이지를 벗어나지 마세요. 분석 도중 페이지 이동 시 진행이 중단됩니다. 세션 타이머를 확인하여 충분한 시간이 남아있는지 체크하세요.
⏱️ 소요 시간: 키워드 100개 기준 약 5~10분 소요됩니다. 분석 중 다른 작업을 하셔도 됩니다. (같은 브라우저 탭은 유지하세요)
4

필터 & 정렬

조건에 맞는 황금 키워드를 쏙쏙 골라내세요

사용 가능한 필터 항목

PC 검색량 최소 ~ 최대 범위 설정
모바일 검색량 최소 ~ 최대 범위 설정
총 검색량 합산 검색량 필터
블로그 발행량 경쟁도 파악

정렬 기능 활용법

결과 테이블의 컬럼 헤더를 클릭하면 해당 항목 기준으로 오름차순/내림차순 정렬됩니다.

  • 검색량 높은 순 — 트래픽이 큰 키워드 우선 확인
  • 경쟁도 낮은 순 — 블로그 발행량이 적은 블루오션 키워드 발굴
  • CPC 높은 순 — 광고 수익이 높은 키워드 식별
🏆 황금 키워드 공식: 검색량 1,000 이상 + 블로그 발행량 10,000 이하 = 경쟁은 낮고 수요는 높은 황금 키워드!
5

세션 타이머 NEW

로그인 유지 시간을 실시간으로 확인하고 연장하세요

세션 타이머란?

로그인 후 헤더에 표시되는 카운트다운 타이머입니다. 세션이 만료되기 전에 남은 시간을 실시간으로 확인할 수 있어, 갑작스러운 로그아웃을 방지합니다.

1:45:30 여유 (30분 초과)
25:10 주의 (30분 이하)
3:45 위험 (5분 이하, 깜빡임)
1

타이머 위치 확인

PC: 상단 헤더 오른쪽의 레벨/잔여일 옆에 표시됩니다.
모바일: 상단 앱 헤더의 레벨 정보 옆에 표시됩니다.
로그인 상태에서만 나타납니다.

2

색상별 의미 이해

초록색 30분 이상 남음 (여유로움)
노란색 30분 이하 (주의 필요, 연장 권장)
빨간색 깜빡임 5분 이하 (즉시 연장하세요!)

3

세션 연장 방법

타이머를 클릭(터치)하면 세션이 즉시 2시간으로 리셋됩니다. 타이머가 살짝 커졌다 돌아오는 애니메이션으로 연장이 확인됩니다.

4

만료 시 자동 처리

타이머가 0에 도달하면 자동으로 로그아웃 페이지로 이동합니다. 작업 내용이 사라지지 않도록 미리 연장하세요!

⚠️ 중요: 자동 분석 중에는 반드시 타이머를 확인하세요. 세션이 만료되면 분석이 중단됩니다. 분석 시작 전 타이머를 클릭하여 2시간으로 리셋하는 것을 권장합니다.
💡 Tip: 페이지를 이동하면(메뉴 클릭 등) 세션이 자동으로 갱신됩니다. 타이머 클릭은 같은 페이지에 오래 머물 때 유용합니다.

API 연동 설정

네이버 SearchAd API와 Blog API를 연결하여 분석력을 극대화하세요

SearchAd API 연동 (Lv.3으로 업그레이드)

1

네이버 검색광고 가입

searchad.naver.com에 접속하여 네이버 검색광고 계정을 생성합니다. (무료)

2

API 키 발급

검색광고 관리 화면 > 도구 > API 사용 관리에서 API 라이선스를 생성합니다. API Key, Secret Key, Customer ID 3가지를 확인하세요.

3

마이페이지에서 입력

키워드마스터 마이페이지에서 3가지 값을 입력하면 즉시 Lv.3 (쿼터 30회)으로 업그레이드됩니다.

Blog API 연동 (Lv.4로 업그레이드)

네이버 개발자센터에서 Blog Search API 키를 발급받고, 마이페이지에 입력하면 Lv.4 (쿼터 60회)으로 업그레이드됩니다.

💡 Tip: API 연동은 완전 무료입니다. 네이버에서 제공하는 무료 API를 활용하므로 추가 비용이 발생하지 않습니다.

검색 히스토리

과거 검색한 키워드를 다시 확인하고 추세를 파악하세요

히스토리 페이지 활용법

  • 최근 검색 키워드 목록 — 언제, 어떤 키워드를 검색했는지 시간순으로 확인
  • 재검색 — 과거 키워드를 클릭하면 최신 데이터로 다시 검색
  • 추세 비교 — 같은 키워드의 검색량 변화를 시간대별로 비교 가능
📊 활용법: 계절별 키워드 분석에 유용합니다. 예를 들어 "에어컨 청소"를 봄에 검색하고, 여름에 다시 검색하면 검색량 변화 추이를 파악할 수 있습니다.

제휴(레퍼럴) 프로그램

친구를 초대하고 서로 보너스 일수를 받으세요

1

레퍼럴 코드 확인

Lv.2 이상 회원이라면 레퍼럴 페이지에서 나만의 8자리 레퍼럴 코드를 확인할 수 있습니다.

2

친구에게 코드 공유

카카오톡, SNS, 블로그 등에서 내 레퍼럴 코드를 공유하세요.

3

친구가 마이페이지에서 코드 입력

친구가 키코드 등록 시 레퍼럴 코드를 입력하면 양쪽 모두 +10일 보너스가 자동 적용됩니다!

레퍼럴 혜택 요약

🎁
추천인 (나)
유효기간 +10일
🎉
피추천인 (친구)
유효기간 +10일
⚠️ 유의사항: 레퍼럴 코드는 계정당 최초 1회만 사용 가능합니다. 여러 명의 친구가 내 코드로 등록하는 것은 횟수 제한 없이 가능합니다.

모바일 앱 활용법

스마트폰에서도 동일한 기능을 사용할 수 있습니다

홈 화면에 추가하기 (PWA)

키워드마스터는 앱 설치 없이 홈 화면에 추가하여 네이티브 앱처럼 사용할 수 있습니다.

iOS

iPhone / iPad

Safari에서 접속 > 하단 공유 버튼 > "홈 화면에 추가" 선택

AOS

Android

Chrome에서 접속 > 상단 메뉴(⋮) > "홈 화면에 추가" 또는 자동 팝업에서 설치

모바일 전용 UI 안내

  • 하단 탭 바 — 검색, 가이드, 가격, 마이, 더보기 5개 탭으로 빠른 이동
  • 상단 헤더 — 현재 페이지명, 레벨, 잔여일, 세션 타이머 표시
  • 더보기 시트 — 히스토리, 레퍼럴, 관리자 등 추가 메뉴를 아래에서 위로 슬라이드
  • 다크 모드 — 상단 달/해 아이콘으로 전환 (눈이 편안합니다)

모바일 세션 타이머

모바일에서도 상단 헤더에 세션 타이머가 표시됩니다. 타이머 터치로 세션을 연장할 수 있어, 외출 중에도 세션 만료 걱정 없이 사용하세요.

📱 Tip: PC에서 모바일 버전으로 전환하려면 URL에 &view=mobile을 추가하세요. 반대로 &view=pc로 데스크톱 버전을 볼 수 있습니다.
