🤖 AI 키워드 생성 (프리미엄 기능)

AI에게 키워드를 요청하면, 200~500개의 관련 키워드를 자동 생성합니다.

사용법:
1. 왼쪽 사이드바의 "AI 키워드 생성" 섹션에서 키워드 입력 (예: "강남 맛집")
2. AI 제공자 선택 (GPT, Gemini, Claude, Grok, Perplexity)
3. 생성 개수 선택 (30~500개, 기본 200개)
4. "AI 생성" 버튼 클릭
5. ➔ 생성 완료 즉시 자동으로 대량수집이 시작됩니다!
💡 팁: AI가 만든 키워드 200개가 바로 대량수집 입력란에 들어가고, 연관확장 + 자동완성 + 원문포함 3가지가 자동으로 켜진 상태로 무한수집이 시작됩니다. 앉아서 구경만 하면 됩니다!

⚡ 대량수집 시스템

입력된 키워드를 하나씩 자동 검색하며 네이버 검색광고 데이터를 수집합니다.

수집 방법:
1. "대량 수집 & 분석" 섹션에 시드 키워드를 줄바꿈으로 입력
2. 아래 3가지 토글 옵션을 설정
3. 수집 버튼 클릭: 1,000개 / 10,000개 / ♾ 무한수집
4. 실시간 라이브 피드에서 수집 현황을 확인!
🎯 3가지 핵심 토글 옵션 (무한 확장의 비밀)
🔗

연관확장

네이버 검색광고 API가 돌려주는 연관 키워드를 자동으로 수집 큐에 추가합니다.

예시: "강남 맛집" 검색 ➔ API가 "강남역 맛집", "신논현 맛집", "강남 점심" 등 연관 키워드를 반환 ➔ 이 키워드들이 자동으로 큐에 추가되어 다시 검색됨 ➔ 끝없이 확장!
🔍

자동완성

네이버 자동완성 API에서 사람들이 실제로 검색하는 키워드를 가져와 큐에 추가합니다.

예시: "강남 맛집" 입력 ➔ 자동완성에서 "강남 맛집 추천", "강남 맛집 데이트", "강남 맛집 소개팅" 등이 나옴 ➔ 이것들도 큐에 추가 ➔ 롱테일 키워드 대량 발굴!
🎯

원문포함

내가 처음에 입력한 시드 키워드 글자가 포함된 키워드만 수집합니다. 관련없는 키워드를 걸러냅니다.

예시: 시드="강남 맛집" ➔ "강남역 맛집 추천" (✔ 포함) / "서초 레스토랑" (✘ 제외)
끄면: 연관 키워드를 전부 수집 (범위 넓어짐)
💡 작동 원리: 키워드 A를 검색 ➔ 연관 키워드 B,C,D 발견 ➔ 자동완성에서 E,F,G 발견 ➔ 원문포함 필터로 A가 들어있는 것만 남김 ➔ B,C,E를 다시 검색 ➔ 또 새로운 키워드 발견 ➔ 무한 반복!
⚠ 속도가 너무 빠르면 자동으로 느려지고 빨간색 경고 토스트가 뜹니다. 걱정 마세요, 성공하면 다시 빨라집니다!

📈 필터 & 정렬로 황금 키워드 찾기

수집된 수천 개의 키워드 중에서 원하는 조건의 키워드만 골라냅니다.

필터 옵션 설명:

💻 PC조회↑ — PC에서 이 키워드를 한 달간 검색한 횟수의 최솟값
📱 MO조회↑ — 모바일에서 한 달간 검색한 횟수의 최솟값
📊 합계↑ / 합계↓ — PC+MO 합산 검색량의 최솟값 / 최댓값
예) 합계↑ 100, 합계↓ 5000 ➔ 월 검색량 100~5000 사이만 수집
📝 블로그↑ — 네이버 블로그 글 수의 최솟값
🖱 클릭↑ — 월 평균 클릭 수의 최솟값
💰 입찰↑ — 광고 입찰가(원) 최솟값. 높을수록 돈이 되는 키워드!
🌟 초보자 추천: 합계↑에 1만 입력하세요. 검색량이 0인 쓸모없는 키워드만 걸러줍니다!
💡 정렬: 필터 결과 테이블의 컬럼 헤더(키워드, PC조회, MO조회, 합계, 블로그, 입찰1위)를 클릭하면 오름차순/내림차순 정렬됩니다.

💡 실전 활용 사례
실제로 키워드마스터를 활용하는 방법을 알려드립니다.

🎯 블로그 글감 찾기

관심 분야의 키워드를 검색하여 검색량이 높으면서 경쟁이 낮은 키워드를 찾습니다.

예시: "다이어트" 검색 → "저탄고지 다이어트 부작용" (검색량 5,000, 경쟁 낮음) 발견 → 해당 주제로 포스팅 작성 → 상위 노출 성공

💰 광고비 절감하기

같은 의미지만 CPC가 낮은 키워드를 찾아 광고 효율을 높입니다.

예시: "영어학원" (CPC 3,000원) → "성인 영어회화" (CPC 1,200원) 발견 → 광고비 60% 절감하면서 같은 효과

📈 트렌드 선점하기

실시간/인기 키워드에서 상승세인 키워드를 미리 선점합니다.

예시: "캠핑용품" 검색 → 4월부터 검색량 급증 확인 → 3월에 미리 콘텐츠 준비 → 시즌 시작과 동시에 대량 유입

🔎 롱테일 키워드 발굴

연관 키워드에서 구체적이고 검색량이 적당한 키워드를 찾습니다.

예시: "노트북" (경쟁 높음) → "대학생 가성비 노트북 추천" (검색량 2,000, 경쟁 낮음) → 틈새 공략 성공
💬 자주 묻는 질문
네이버 광고 시스템의 공식 API를 통해 월간 검색량, PC/모바일 검색 비율, 경쟁 정도(낮음/중간/높음), 클릭당 비용(CPC), 연관 키워드, 스코어 등급(S~F) 등을 제공합니다.
