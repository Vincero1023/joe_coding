줄바꿈, 콤마, 세미콜론으로 여러 키워드를 나눌 수 있습니다.

template
제목 생성 모드
템플릿 안내

템플릿 모드는 현재 선택된 골든 키워드로 네이버형/블로그형 제목 세트를 바로 만듭니다. 별도 API 설정 없이 가장 빠르게 사용할 수 있습니다.

제목 키워드 조합

현재 생성 대상: 단일 + V1 + V2 + V3 / 단일은 안전형, V1은 선별 기반, V2는 탈락 연계 확장, V3는 저검색량 실험형입니다.

창의성 프리셋

(추천) 균형형: 제목 다양성과 품질 안정성의 균형이 가장 무난한 기본값입니다. (temperature 0.7)

AI 프롬프트

새 탭에서 제목 생성용 추가 지침을 수정합니다.

기본 시스템 프롬프트만 사용 중입니다.

API 키는 서버에 저장하지 않고 현재 브라우저 localStorage 에만 보관합니다.
