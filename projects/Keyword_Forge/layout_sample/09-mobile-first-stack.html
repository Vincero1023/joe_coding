How It Works

모바일 구조 설명

  • 첫 화면오늘의 실행과 빠른 입력만 먼저 보여줍니다.
  • 두 번째 층실시간 선별 결과가 카드로 올라옵니다.
  • 하단 내비게이션결과, 선별, 보관함, 설정을 이동합니다.

Preserved Features

기능 보존 방식

  • 빠른 입력은 홈 상단 카드
  • 2축 선별은 별도 탭 화면
  • 제목 설정은 하단 시트
  • 기록/보관함/예약은 더보기 허브
