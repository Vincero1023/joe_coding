# Layout Sample Gallery

`layout_sample` 폴더는 현재 Keyword Forge의 기능 개념을 유지한 채, 화면 정보 구조와 배치를 완전히 다르게 실험해 보는 정적 목업 모음입니다.

중요한 전제:

- 실제 API 호출이나 버튼 동작은 연결하지 않았습니다.
- 대신 현재 앱의 핵심 기능 덩어리를 모두 화면 설계 안에 유지했습니다.
- 기준 기능은 `빠른 입력`, `모드별 실행`, `단계별 결과`, `2축 선별`, `제목 생성`, `운영 설정`, `기록/보관함/예약`, `진단/로그`입니다.

포함된 레이아웃:

1. `01-command-center.html`
   빠른 실행과 실시간 작업대를 중심으로 한 3분할 지휘실
2. `02-pipeline-kanban.html`
   단계별 진행을 한눈에 보는 칸반 파이프라인 보드
3. `03-operator-cockpit.html`
   운영 지표와 예외 상태를 앞세운 실무용 조종석
4. `04-notebook-studio.html`
   노트 앱처럼 읽고 조정하는 문서형 작업실
5. `05-modular-grid.html`
   고밀도 위젯을 빠르게 훑는 모듈러 대시보드
6. `06-timeline-flow.html`
   수집부터 제목까지 흐름을 따라가는 세로형 타임라인
7. `07-queue-ops-board.html`
   예약, 루틴, 보관함 재활용에 초점을 둔 운영 보드
8. `08-title-lab.html`
   제목 생성과 프롬프트 조합을 주인공으로 둔 랩
9. `09-mobile-first-stack.html`
   모바일 우선 구조를 가정한 카드 스택형 레이아웃
10. `10-briefing-room.html`
    오늘 해야 할 일과 우선순위를 빠르게 정리하는 브리핑 룸

시작점:

- `index.html`을 열면 10개 레이아웃 요약과 링크를 볼 수 있습니다.
